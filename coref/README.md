# Coreference Prompt Experiments

This repository contains small-batch experiments evaluating
cluster-based coreference resolution using LLM prompting.

## Current Status

- Dataset: Artificial sanity-check batch
- Evaluation: MUC, B³, CEAFₑ, CoNLL F1
- Setup: ChatGPT web (no API)

## Next Steps

- Add harder test cases
- Compare prompt variants
- Introduce real datasets (PreCo)

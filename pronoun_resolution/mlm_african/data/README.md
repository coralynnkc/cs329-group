# African Benchmark Raw Data

This folder contains the raw benchmark-family CSVs for the African pronoun-resolution experiment.

Files included:

- `en_master.csv`
- `am_master.csv`
- `ig_master.csv`
- `zu_master.csv`
- `*_questions_only.csv`
- `*_answers_only.csv`

These are the inputs used to derive the canonical split files under:

- `../en/source/`
- `../am/source/`
- `../ig/source/`
- `../zu/source/`

The raw CSVs are kept here so they stay attached to the `mlm_african/` experiment rather than cluttering the root of `pronoun_resolution/`.

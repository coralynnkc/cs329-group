# Pronoun Resolution — Prompt Testing

This directory contains the reproducible split-generation pipeline for evaluating
pronoun-resolution prompts.

## Quick Start

```bash
cd pronoun_resolution/testing
python generate_splits.py          # run with defaults
python generate_splits.py --help   # see all options
```

## Directory Layout

```
testing/
  generate_splits.py   # pipeline script (see docstring for full docs)
  testing.md           # this file
  splits/              # output CSVs (generated by the script)
  metadata/            # split_summary.json and split_summary.md
```

## Splits

| File | Purpose |
|------|---------|
| `english_master_normalized.csv` | Full normalised dataset with difficulty scores |
| `fewshot_bank.csv` | Exemplars for few-shot prompts — **never scored** |
| `dev.csv` | Primary 100-item eval set (stratified by label + length) |
| `challenge.csv` | 50-item stress-test set (highest difficulty proxy) |
| `holdout.csv` | Remainder — do not use until final prompt selection |
| `split_manifest.csv` | One row per item: item_id, split, gold_answer, difficulty_score |

See `metadata/split_summary.md` for schema decisions and construction notes.
